# Mac App Store Screenshot Requirements

## Required Sizes (at least one of each)

### For all Macs (Required)
| Size | Description |
|------|-------------|
| **2880 x 1800** | Retina MacBook Pro 15" |
| **2560 x 1600** | Retina MacBook Pro 13" |

### Optional (but recommended)
| Size | Description |
|------|-------------|
| 1440 x 900 | Non-Retina displays |
| 1280 x 800 | Minimum size |

## Format Requirements
- **Format:** PNG (preferred) or JPEG
- **Color space:** sRGB
- **No alpha/transparency**
- **No rounded corners** (Apple adds these)
- **No device frames** (Apple adds these optionally)

## Number of Screenshots
- **Minimum:** 1 screenshot
- **Maximum:** 10 screenshots per localization
- **Recommended:** 4-6 screenshots showing key features

## Suggested Screenshots for MacCoolClean

1. **Main Interface** - Show the full app with sidebar and scanned files
2. **Scanning in Progress** - Show the scanning animation
3. **Large Files Found** - Show results with file sizes
4. **File Details** - Show the tooltip/details popup
5. **Menu Bar Widget** - Show the menu bar extra
6. **Upgrade Screen** - Show the Pro upgrade modal

## How to Take Screenshots at Correct Resolution

### Option 1: Set Display Resolution (Recommended)
1. System Settings → Displays
2. Set to "More Space" (higher resolution)
3. Resize app window to fill screen
4. Take screenshot with ⌘⇧3 (full screen) or ⌘⇧4 (selection)

### Option 2: Use Simulator or Resize
Use Preview or an image editor to resize to exact dimensions.
